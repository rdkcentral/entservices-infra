# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-03-23

### Added
- Initial release of GstreamerService plugin
- Support for starting GStreamer pipelines with custom configurations
- Support for stopping GStreamer pipelines
- Pipeline status querying functionality
- Event notifications for pipeline state changes and errors
- Out-of-process plugin architecture for stability
