## SharedStorage Plugin Migration

**Note**: The SharedStorage plugin has been migrated to its own standalone repository as of February 2026.

- **New Repository**: [entservices-SharedStorage](https://github.com/rdkcentral/entservices-SharedStorage)
- **Reason**: Better modularity and independent versioning
- **Impact**: No functional changes; SharedStorage continues to work as before
- **For existing deployments**: Update your build scripts to include both repositories if SharedStorage is needed
